# Camera Tools

Scripts for controlling and enhancing camera behavior in After Effects.

## Scripts

### Camera Focus Controller V1.2
Adds an advanced focus controller to the selected camera layer. Creates a 3D null object that controls the camera's focus distance, making it easy to animate focus points in 3D space.

**Usage:** Select a camera layer and run the script. A "FOCUS CONTROL" null will be created that you can position to control camera focus.

**Features:**
- Automatic Depth of Field enablement
- 3D null object for visual focus control
- Works with both one-node and two-node cameras
