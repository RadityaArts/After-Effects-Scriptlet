# Compositions

Scripts for managing, organizing, and enhancing compositions in After Effects.

## Scripts

### [Auto-Fit Precomp](Auto-Fit%20Precomp.jsx)
Automatically precomposes selected layers with smart cropping to content bounds. Creates perfectly sized precomps that eliminate excess transparent space while preserving all animations and layer relationships.

**Usage:** Select one or more layers and run the script. Enter a custom name or leave blank to auto-name.

**Features:**
- Automatic crop to content boundaries
- Preserves keyframes, expressions, and parenting
- Accounts for effects (blur, glow, shadow, stroke)
- Handles masks, 3D layers, and shape strokes
- Single or multi-layer group precomposing
