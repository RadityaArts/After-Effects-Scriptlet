# Layers

Scripts for controlling and enhancing layer behavior in After Effects.

## Scripts

### [Center Layer](Center%20Layer/)
Centers the anchor point and positions selected layers at the center of the composition.

**Usage:** Select one or more layers and run the script.

**Features:**
- Centers anchor point to layer content
- Positions layer at composition center
- Preserves Z-position for 3D layers
- Works with multiple layers simultaneously

---

### [Fit to Comp Height](Fit%20to%20Comp%20Height/)
Automatically scales selected layers to fit the composition height while maintaining aspect ratio.

**Usage:** Select one or more layers and run the script.

**Features:**
- Scales layers to match composition height
- Centers layers in composition
- Accounts for layer rotation
- Supports layers with masks
- Preserves keyframes

---

### [Fit to Comp Width](Fit%20to%20Comp%20Width/)
Automatically scales selected layers to fit the composition width while maintaining aspect ratio.

**Usage:** Select one or more layers and run the script.

**Features:**
- Scales layers to match composition width
- Centers layers in composition
- Accounts for layer rotation
- Supports layers with masks
- Preserves keyframes
